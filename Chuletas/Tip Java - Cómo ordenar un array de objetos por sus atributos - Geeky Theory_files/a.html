                                    Cristian • hace 4 años

                                    usar solamente un compareTo complica un poco las cosas si necesitas tener alternativas para comparar, algo interesante es usar un comparator y después ordenar la lista usando ese comparador.

                                    ejemplo

                                    Comparator alturaCmp = (Comparator<persona>) (Persona p1, Persona p2) -> {
                                    if(p1.altura() > p2.altura) return 1;
                                    if(p1.altura() < p2.altura) return -1;
                                    return 0;
                                    };

                                    y luego usar Collections utils

                                    Collections.sort(listaPersonas, alturaCmp);

                                    y de esa forma ordenamos por altura, también podemos crear otro comparador y ordenar por otro atributo

                                    por ejemplo edadCmp y usar el campo edad de la persona.

                                    saludos!
